About Me

Computer Engineer

Hey there! I'm Aninda. I graduated from City College of New York with a Bachelor's in Computer Engineering. I have been fascinated by computer technology ever since my father got me my first computer with Windows 2000 Operating System installed. I loved to stress test the operating system and figure out the true capabilities of the computer. I took my first coding class of CS101 at CCNY and it was an overwhelming experience. I struggled to understand vectors and pointers. At the same time, I was having a hard time editing my code in VIM and compiling my CPP file in Arch Linux. I ended up dropping the class just to enroll in the same class with the same professor the next semester. Since then I have worked on many academic and personal projects.
One of my favorite projects is called Safe Maps and it was a year-long senior design group project. Safe Maps, an ios-app built with react native and node, is a navigation app that generates safe routes based on historical crime datasets. I have worked on the front end and with the data extensively in this project and this project is what drove me to pursue web development. Then I joined Workschool, a start-up vowing to structure online courses and make them more accessible to learners all around the world. I worked there as a Fullstack Developer and learned various tools on the job such as Typescript, NextJs, Google Auth, AWS deployments etc. I am always eager to learn and experiment with new tools and technologies and am passionate about working towards a better future.
